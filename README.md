# Portfolio

Premium personal portfolio website with modern UI/UX, inspired by Vercel, Linear, and ReactBits.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Shadcn UI
- Lucide Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
/app
  layout.tsx
  page.tsx
  globals.css
  projects/page.tsx
  contact/page.tsx

/components
  ui/          # Shadcn + MagneticButton
  layout/      # Navbar, Footer, AboutSection, ContactSection
  hero/        # Hero
  projects/    # ProjectCard, ProjectsSection
  skills/      # SkillsGrid

/lib
  utils.ts
```

## Build

```bash
npm run build
npm start
```
